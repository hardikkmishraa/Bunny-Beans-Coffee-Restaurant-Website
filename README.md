# Bunny-Beans-Coffee-Restaurant-Website
Bunny Beans is a single-page coffee restaurant website built using HTML, CSS, and JavaScript to practice responsive design and frontend development.

## 1. Introduction

I developed a **single-page coffee restaurant website** named **“Bunny Beans”** 

The purpose of this project is to design an attractive and responsive website using **basic frontend technologies**.

## 2. Objective of the Project

The main objectives of this project are:

- To design a visually attractive restaurant website
- To practice **HTML, CSS, and JavaScript**
- To understand responsive web design
- To improve UI/UX and layout designing skills

## 3. Technologies Used

- **HTML5** – for website structure
- **CSS3** – for styling, layout, and responsiveness
- **JavaScript** – for basic interactivity

No frameworks or libraries are used in this project.

## 4. Project Structure

```
bunny-beans/
├── index.html   (Main webpage)
├── style.css    (Design and layout)
├── script.js    (JavaScript functionality)
└── README.md    (Project description)
```

## 5. Description of Website Sections

The website includes the following sections:

- **Hero Section** – Introduction of the café with heading and buttons
- **About Section** – Information about Bunny Beans café
- **Menu Section** – Displays the café menu using an image
- **Signature Specials** – Shows special coffee items
- **Gallery Section** – Images of coffee and café ambience
- **Testimonials** – Customer reviews using a slider
- **Contact Section** – Address, timings, and Google Map
- **Footer** – Café name and copyright text

## 6. Features of the Website

- Fully responsive design
- Smooth scrolling navigation
- Sticky navigation bar
- Image gallery with hover effects
- Testimonial slider using JavaScript
- Simple and clean user interface

## 7. How to Run the Project

1. Download or copy all files into a single folder
2. Open `index.html` in any web browser
3. The website will run without any installation

- How to design responsive layouts
- How to add basic animations and interactions
- How to organize frontend project files properly
